# 간단한 메모장

브라우저에서 바로 여는 간단한 메모장입니다. 주요 기능:

- 자동저장 (입력 후 0.8초)
- `Ctrl/Cmd+S`로 저장
- `Ctrl/Cmd+O`로 로컬 텍스트 파일 열기
- `Ctrl/Cmd+N`으로 새 문서
- 텍스트 파일로 내보내기
- 다크 모드 토글

사용 방법:

1. 로컬에서 `index.html`을 브라우저로 엽니다.
2. 입력하면 자동으로 저장됩니다 (localStorage 사용).
3. 필요하면 내보내기 버튼으로 `.txt` 파일을 받을 수 있습니다.

문제가 있거나 기능 추가 원하면 알려주세요.
